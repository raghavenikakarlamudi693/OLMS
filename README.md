# Online Leave Management System (OLMS)

A web-based system for managing employee leave requests, built with Java, JSP, Servlets, and JDBC for database connectivity.

## Features
- Employee can apply for leave
- HR can approve or reject leave
- View leave history
- Database operations via JDBC

## Technology Stack
- Java (Servlets, JSP)
- JDBC for database connectivity
- Eclipse IDE
- Apache Tomcat Server (for running the web app)
- GitHub for version control
- Oracle (for data storage)
